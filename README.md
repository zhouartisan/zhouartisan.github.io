# 一位图书管理猿

个人博客源码，线上访问：**[https://zhouartisan.github.io](https://zhouartisan.github.io/)**

众鸟欣有托，吾亦爱吾庐。主要记录读书笔记、工作与工具、以及一点生活随感。

## 技术说明

本站基于静态站点生成器 [Jekyll](https://jekyllrb.com/)，使用主题 [**Chirpy**](https://github.com/cotes2020/jekyll-theme-chirpy/)，通过 [GitHub Actions](https://github.com/features/actions) 构建并发布到 GitHub Pages。

文章放在 `_posts` 目录；站点标题、签名、头像等在 `_config.yml` 中配置。

## 本地预览（可选）

```bash
bundle install
bundle exec jekyll serve
```

浏览器打开 <http://127.0.0.1:4000> 。
